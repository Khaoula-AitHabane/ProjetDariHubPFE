# 🏠 Home Services Platform - PFE Project

## 📌 Description
This project is a Final Year Project (PFE) that aims to develop a web platform for home services.  
The platform connects clients with service providers in an easy and efficient way.

The system is divided into two main parts:
- 🔙 Backend: Developed with Laravel (REST API)
- 🎨 Frontend: Developed with React (User Interface)

---

## 🚀 Features

### 👤 Users
- Client registration and login
- Service provider registration
- Admin dashboard

### 🏡 Services
- House rental service (Dyore)
- Furniture rental service (Mafrouchat)
- Home services (cleaning, plumbing, electricity, gardening...)

### 🔎 Functionalities
- Search for services
- Booking system
- User management
- Service management
- Authentication system

---

## 🛠️ Technologies Used

### Backend
- Laravel
- MySQL
- REST API

### Frontend
- React
- JavaScript
